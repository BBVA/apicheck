What's apitest
==============

A Docker analysis tools

Very quick install
==================

**Install from Pypi**

.. code-block:: bash

    > python3.5 -m pip install apitest

**From source**

.. code-block:: bash

    > git clone https://github.com/bbva/apitest.git apitest
    > cd apitest
    > python3.5 setup.py install

Show options:

.. code-block:: bash

    > apitest -h
